### Building the Data Warehouse (Data Engineering)

#### Objective 
To develop a modern data warehouse using SQL Server to consolidate sales data for analytics and decision-making.

#### Specifications:
- **Data Sources:** Import CSV files from ERP and CRM systems.
- **Data Quality:** Cleanse and resolve data quality issues before analysis.
- **Integration:** Combine both data sources into a single, user-friendly data model for analytical queries.
- **Scope:** Focus on the latest dataset only; historical data is not required.
- **Documentation:** Provide clear documentation of the data model for business and analytics stakeholders.
